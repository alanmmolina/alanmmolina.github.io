Projeto desenvolvido em Novembro de 2018 como requisito para aprovação na disciplina de Circuitos Eletrônicos Analógicos. O trabalho proposto pelo docente consistiu no emprego de circuitos analógicos para adequação de sinais; nesse sentido, foi desenvolvido um sensor capaz de identificar cores cujo princípio de operação foi baseado no modo como o olho humano diferencia as cores do espectro. Além disso, para demonstrar uma aplicação do sensor em questão, utilizou-se uma versão ligeiramente modificada da plataforma dimensionada no projeto de mini robôs do grupo Mashina Robotics.
